# Snippet Files

Copy-paste prompt contract snippets, taken directly from
`src/prompt_preflight/data/prompt_templates.json`.

Each file has the Markdown, XML, and TOML version of that profile's contract.
Open the file for the profile you need and copy the block for the format you
want into your prompt.

Available: `software.md`, `image.md`, `research.md`, `data_analysis.md`.
